# 📊 Superstore Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Sales Performance Dashboard created using **Power BI**. The dashboard helps analyze sales, profit, quantity, and order performance across different categories, regions, and shipping modes.

The main goal of this project is to convert raw sales data into meaningful insights that can help businesses understand their performance and make better decisions.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Compare sales and profit across different categories.
* Identify the best-performing products.
* Understand sales distribution by region and ship mode.
* Create an interactive dashboard for business analysis.

---

## 🛠️ Tools Used

* Microsoft Power BI
* Power Query
* DAX
* Excel / CSV Dataset

---

## 📂 Dataset Information

The dataset contains sales transaction details, including:

* Order ID
* Region
* State
* Category
* Sub-Category
* Ship Mode
* Sales
* Profit
* Quantity

---

## 📈 Dashboard KPIs

The dashboard displays:

* 💰 Total Sales
* 📈 Total Profit
* 📦 Total Quantity Sold
* 🛒 Total Orders

---

## 📊 Dashboard Features

The dashboard includes:

* Sales by Category
* Profit by Segment
* Sales and Profit Comparison
* Top 10 Sub-Categories by Sales
* Region Filter
* Ship Mode Filter
* Interactive KPI Cards

---

## 🔍 Key Insights

* The dashboard helps compare sales and profit across different categories.
* Users can filter the report by Region and Ship Mode.
* The Top 10 Sub-Categories chart highlights the best-selling products.
* KPI cards provide a quick overview of business performance.
