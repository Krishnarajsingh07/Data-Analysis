# 📊 YouTube Data Analysis Dashboard

## 📌 Project Overview
This project analyzes YouTube channel performance using SQL, Python, and Power BI. The dataset was cleaned, explored, and visualized to identify trends in subscribers, video views, earnings, uploads, channel categories, and country-wise performance.

The project demonstrates the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and business insights.

---

## 🎯 Objectives

- Analyze YouTube channel performance.
- Identify the top-performing channels based on subscribers, views, and earnings.
- Compare channel performance across different countries and categories.
- Perform SQL-based business analysis.
- Build an interactive Power BI dashboard.

## 📌 Key Insights

- Entertainment channels dominate subscriber counts.
- The United States and India have the highest number of top-performing channels.
- A few channels contribute a significant share of total subscribers.
- Higher uploads do not always result in higher earnings.
- Subscriber growth varies significantly across channel categories.
