# Data-Analysis
An interactive Power BI dashboard built using the Titanic dataset to analyze passenger survival, demographics, and key insights through data visualization



## About This Project

This is my Power BI dashboard project based on the Titanic dataset.

In this project, I analyzed the passenger data and created an interactive dashboard to understand the survival of passengers. I used different charts and KPIs to find useful insights from the data.

This project helped me improve my Power BI skills and understand how to present data in a simple and meaningful way.

---

# Tools Used

* Microsoft Power BI
* Power Query
* DAX
* CSV Dataset

---

# Dataset

The dashboard is created using the Titanic dataset.

The dataset contains information such as:

* Passenger ID
* Passenger Class
* Gender
* Age
* Fare
* Embarked Port
* Survival Status

---

# Dashboard Features

The dashboard includes:

* Total Passengers Card
* Total Fare Card
* Survival Status Pie Chart
* Passenger Class Slicer
* Gender Distribution Donut Chart
* Survival Count Column Chart
* Survival by Gender (100% Stacked Column Chart)
* Passenger Details Table (Age Group and Embarked)

---

# Key Insights

From this dashboard, I observed:

* The dashboard shows the total number of passengers.
* It displays the total fare collected.
* Survival status can be compared easily using the pie chart.
* The gender distribution of passengers is shown using a donut chart.
* Survival can be compared by gender using the stacked column chart.
* Passenger Class slicer allows users to filter the dashboard interactively.
* The table shows passenger count according to Age Group and Embarked location.
